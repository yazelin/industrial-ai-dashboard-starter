# Industrial AI Dashboard Starter CI Design

## Positioning

**Audience:** 給自動化、AGV、PLC、工業 IoT 工程師  
**Promise:** 先用模擬 AGV / machine data 做出 dashboard，再替換成 MQTT、ROS2、Modbus 或 PLC API。  
**Category cue:** AGV / WebSocket / industrial data

## Visual Identity

- **Primary accent:** `#fb923c`
- **Secondary accent:** `#0284c7`
- **Background:** `#0b111b`
- **Graphic motif:** `factory`
- **Style:** dark developer-tool landing page, technical grid, luminous product glyph, high-contrast CTA.

## Palette

- `#fb923c`
- `#0284c7`
- `#111827`
- `#ffedd5`

## Typography

- Display/UI: Inter
- Technical labels/code: JetBrains Mono
- Headline: large, tight tracking, high contrast
- CTA and labels: bold, concise, action-oriented

## Assets

- `assets/banner.svg`: Open Graph / README / hero banner
- `assets/logo.svg`: square product mark
- `index.html`: GitHub Pages CTA landing page

## CTA Strategy

Primary CTA collects email through FormSubmit and tags submission with `industrial-ai-dashboard-starter`.
Secondary CTA sends users to GitHub repo/template.
