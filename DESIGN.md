# 工業 AI Dashboard 入門模板 CI Design

> English name: Industrial AI Dashboard Starter

## 定位

**主要受眾：** 適合自動化、AGV、PLC、工業 IoT 工程師，先做出可展示的 Web 監控介面。  
**核心承諾：** 先用模擬 AGV / machine data 做出 dashboard，再替換成 MQTT、ROS2、Modbus 或 PLC API。  
**痛點切入：** 工業案場常卡在展示與整合；這個模板先給你一個能跑的即時監控骨架。  
**類別提示：** AGV / WebSocket / industrial data
